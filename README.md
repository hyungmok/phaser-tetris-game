# Phaser Tetris Game

A classic Tetris game built with Phaser.js.

## How to Play

1.  Clone this repository.
2.  Open `index.html` in your web browser.
3.  Alternatively, you can run a local web server to serve the files.

## Controls

-   **Left Arrow Key**: Move the piece left
-   **Right Arrow Key**: Move the piece right
-   **Down Arrow Key**: Soft drop (speed up the fall)
-   **Up Arrow Key**: Rotate the piece

Enjoy the game!
